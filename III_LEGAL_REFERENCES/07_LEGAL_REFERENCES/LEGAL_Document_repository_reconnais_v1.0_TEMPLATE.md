## üìã QUICK START GUIDE (Option A: Free Route)

### **Step 1: Clone NJS-Firebase-SaaS-Boilerplate**
```bash
git clone https://github.com/WHEREISDAN/NJS-Firebase-SaaS-Boilerplate.git hrbiz-lms
cd hrbiz-lms
npm install
```

### **Step 2: Set Up Firebase**
1. Go to https://console.firebase.google.com
2. Create new project: "HRBizLMS"
3. Enable Firestore, Auth (Email/Password), Storage
4. Copy config to `.env.local`:
```env
NEXT_PUBLIC_FIREBASE_API_KEY=your_key
NEXT_PUBLIC_FIREBASE_AUTH_DOMAIN=your_domain
NEXT_PUBLIC_FIREBASE_PROJECT_ID=your_project
NEXT_PUBLIC_FIREBASE_STORAGE_BUCKET=your_bucket
```

### **Step 3: Add Multi-Tenancy**
Create `/lib/firestore-schema.ts`:
```typescript
export const schema = {
  companies: {
    name: 'string',
    address: 'string',
    employeeCount: 'number',
    consultantId: 'string', // Your user ID
    programs: 'array' // HPP, WVPP
  },
  employees: {
    companyId: 'string',
    firstName: 'string',
    lastName: 'string',
    email: 'string',
    trainingRecords: 'array'
  },
  training_sessions: {
    companyId: 'string',
    programId: 'string',
    date: 'timestamp',
    attendedEmployees: 'array'
  }
}
```

### **Step 4: Add Certificate Generation**
```bash
npm install jspdf
```

Create `/lib/generate-certificate.ts`:
```typescript
import jsPDF from 'jspdf';

export async function generateCertificate(employee, program, session) {
  const doc = new jsPDF({ orientation: 'landscape' });
  // Add certificate content
  // (See blueprint for full implementation)
  return doc.output('blob');
}
```

### **Step 5: Deploy**
```bash
npm run build
vercel deploy
```

**Done!** You now have a working LMS platform.

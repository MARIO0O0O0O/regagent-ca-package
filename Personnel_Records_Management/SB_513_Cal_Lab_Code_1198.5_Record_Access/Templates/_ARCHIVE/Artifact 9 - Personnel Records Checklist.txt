California Personnel Records Audit (2026)
Version: 1.0 Mandates: SB 513, SB 464, Lab Code 1198.5
1. The Container System (Mandatory Segregation)
* [ ] Container 1: General Personnel: Performance, Offer Letters, Training Certs.
* [ ] Container 2: Medical (Confidential): Dr Notes, Leave Certs, ADA. (Strict Access).
* [ ] Container 3: I-9 Immigration: Forms only. (Audit protection).
* [ ] Container 4: Payroll: Tax forms, W-4, Timecards.
* [ ] Container 5: Investigative (Privileged): Harassment complaints, Legal notes.
* [ ] Container 6: Pay Data (SB 464): Demographic data for reporting.
2. SB 513 Training Records (Jan 1, 2026)
Training records are now "Personnel Records".
* [ ] 5 Metadata Fields:
   1. Employee Name.
   2. Provider.
   3. Date/Duration.
   4. Core Competencies (Detailed topics).
   5. Certificate.
* [ ] Retention: 3 years post-termination.
3. Access Matrix
* Employee Access: Can see Container 1 & 4. CANNOT see 2, 3, 5, 6.
* Timeline: 30 days to produce copies upon request.
4. CROSS-REFERENCE
* Harassment: Complaints go in Container 5.
* Leave: Medical notes go in Container 2.
* SB 294: Emergency Arrest Contact goes in Container 5.

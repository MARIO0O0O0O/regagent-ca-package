California Leave of Absence Checklist (50-100 Employees)
Version: 2.0 Applicability: 50+ Employees (Includes FMLA)
1. FMLA Overlay
* [ ] FMLA (50+): 12 weeks federal leave.
* [ ] Concurrent: FMLA runs at the same time as PDL.
* [ ] Interaction:
   * Pregnancy: PDL + FMLA run together. CFRA runs after.
   * Non-Pregnancy: FMLA + CFRA run together.
2. Administration
* [ ] Notices: WH-381 (Eligibility) & WH-382 (Designation) required.
* [ ] Certifications: WH-380-E required.
3. CROSS-REFERENCE: HOW OTHER LAWS IMPACT LEAVE
3.1 PERSONNEL RECORDS → LEAVE
Compliance Impact:
* [ ] FMLA File: Sub-section in Container 2 for FMLA forms. Cross-Reference: → [Personnel Records Checklist]
3.2 WORKERS' COMP → LEAVE
Compliance Impact:
* [ ] Designation: Designate Workers' Comp as FMLA/CFRA to burn entitlement. Cross-Reference: → [Workers' Comp Checklist]
3.3 POSTINGS → LEAVE
Compliance Impact:
* [ ] FMLA Poster: Mandatory for 50+. Cross-Reference: → [Posting Checklist]
4. Penalties
* Federal Suit: FMLA allows federal lawsuits + liquidated (double) damages.

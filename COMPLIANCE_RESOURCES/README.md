# COMPLIANCE RESOURCES
## California Employer Compliance Packages

This section contains structured compliance packages for California employers.
Each package is a self-contained unit including policy documents, training
materials, forms, tools, and recordkeeping guides.

> **Disclaimer:** These materials are compliance resources, not legal advice.
> Employers should consult qualified legal counsel for their specific situation.

## Package Structure

Every package follows this standard structure:

```
N_PACKAGE_NAME/
├── manifest.json       ← Machine-readable metadata (web app source of truth)
├── CHANGELOG.md        ← Version history and update notes
├── README.md           ← Human-readable package overview
├── POLICY/             ← Policy documents
├── FORMS/              ← Forms and templates
├── TRAINING/           ← Training guides and handouts
├── TOOLS/              ← Quick references, decision tools, calculators
└── RECORDKEEPING/      ← Retention schedules and audit guides
```

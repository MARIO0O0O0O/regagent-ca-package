# PACKAGE 02: WORKPLACE VIOLENCE PREVENTION
## California SB 553 Compliance Package
**Version:** 1.0 | **Last Updated:** February 2026

---

## WHAT'S IN THIS PACKAGE

This package provides everything needed to implement and maintain a legally compliant California Workplace Violence Prevention program under SB 553, effective July 1, 2024.

**Training Model:** Live interactive sessions only (in-person or webinar)  
**Training Audience:** ALL employees — single tier, no split  
**Training Frequency:** Annually (every 12 months)  
**Enforcement Agency:** Cal/OSHA (not CRD)

## CRITICAL DIFFERENCES FROM HARASSMENT PREVENTION

| Feature | Harassment Prevention | Workplace Violence |
|---------|----------------------|-------------------|
| Enforcement | CRD (California Civil Rights Dept) | Cal/OSHA |
| Training tiers | Employee (1hr) / Supervisor (2hr) | All employees — same 1hr training |
| Training cycle | Every 2 years | **Every year** |
| Written plan | Policy document | **Formal WVPP required** |
| Incident log | Not required | **Required — 5-year retention** |
| Cal/OSHA inspection | Not applicable | Records must be available on demand |
| Serious injury reporting | Not applicable | Must report to Cal/OSHA within 8 hours |

## IMPLEMENTATION CHECKLIST

**Initial Deployment:**
- [ ] Adopt WVPP (wvpp_v1.0.md) — complete all [COMPANY NAME] fields
- [ ] Designate Plan Administrator ([NAME, TITLE])
- [ ] Conduct initial workplace hazard evaluation (document results)
- [ ] Schedule and deliver initial WVP training for all employees
- [ ] Collect training acknowledgments (WVP-FORM-001) from all employees
- [ ] Activate violent incident log (WVP-FORM-004) — track all incidents

**Annual Requirements:**
- [ ] Deliver annual training to all employees (every 12 months)
- [ ] Complete annual WVPP review (WVP-FORM-005)
- [ ] Update WVPP if operations, locations, or risks have changed
- [ ] Review all incidents from prior year; assess patterns
- [ ] Confirm Cal/OSHA 300 log is current if applicable

**New Hires:**
- [ ] WVP training before or at start of work (best practice)
- [ ] Signed acknowledgment on Day 1
